Teaching: Programming and web-design — interaction¶

In the above graph we see that an early part of python structured depends on Basic HTML (or at least could benafit from what is learnt there). The dependency the other way is small markdownHTML depends on being able to use procedures. However markdownHTML is at the end of web-design (nothing depends on it).

Therefore we should teach web-design first, and move markdownHTML to the end of the programming in python unit.

If we could have a simple command line program to do the markdownHTML, then we could use it much earlier. However while possible on MS-Windows, it is very klunky. Not an easy thing to do, and certenly not easy to teach a class of 20. If we had a Unix system (such as Gnu/Linux), then this would be easy.
